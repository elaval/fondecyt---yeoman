CarrerasController
controller in module fondecytApp

Description

Controller for Carreras explorer

Dependencies

  • $scope
  • CarrerasDataService

Methods

  • updateCarreras()

    Updates the list of valid careers to be displayed (this.carreras) acording to the option this.filtraTopCarreras. If this.filtraTopCarreras is true, it will select a maximum of this.maxCarreras, if not it will display all careers.

    Once updated, will assign the first career to this.selectedCarrera.

  • updateStudentData()

    Updates data for the selected carrera & semestre

Properties

  • colorOptions

    Array with options for colorAttributes

  • colorAttribute

    Selected color attribute

  • data

    Array with student data for the selected career & semester

  • n

    Number of students in the selected data array

  • maxCarreras

    Maximum number of carreras to be displayed when filtraTopCarreras is true

  • semestres

    Array with the semesters options to be chosen

  • selectedSemestre

    Selected semester for data selection

  • psuValido

    Flag to select only data values with a valid psu score (prom_paa>0)

  • loading

    Flag to show a "loading" message when its value is true
