CursosController
controller in module fondecytApp

Description

Controller for Cursos explorer

Dependencies

  • $scope
  • CursosDataService

Methods

  • updateCursosPorUnidadData()

    Updates the data for all the courses corresponing to the selected unit & semester

  • updateEstudiantesPorCursodata()

    Updates the data corresponing to all the students from a particular course & semester

  • updateSemestre()

    Actualiza selección datos de cursos & estudiantes por curso (utilizado luego de un cambio en la selección de semestre)

  • updateUnidadesData()

    Updates the list of valid academic units (unidades academicas). Once updated, will assign the first unit to selectedUnidad and will update the data for all the courses in that unit.

Properties

  • colorAttribute

    Returns
    string

    colorAttribute atributo seleccionado para categorizar por colores

  • colorOptions

    Returns
    array

    colorOptions Listado con opciones de atributo para categorizar por colores

  • cursosPorUnidadData

    Cada registro es un objecto que contiene información agregada por curso (r2, promedio_sakai, ...)

    Returns
    array

    Datos correspondientes a los cursos de la unidad seleccionada

  • estudiantesPorCursoData

    Cada registro es un objecto que contiene información detallada de cada estudiante (actividades_sakai, prom_notas, ...)

    Returns
    array

    Datos correspondientes a los estudiantes del curso seleccionado

  • minEstudiantesPorCurso

    Returns
    int

    Define el número mínimo de estudiantes por curso a ser seleccionado

  • selectedSemestre

    Returns
    string

    Semestre seleccionado

  • semestres

    Returns
    array

    Lista con los semestres disponibles para ser seleccionados

  • unidades

    Returns
    array

    Lista con las unidades académicas disponibles para ser seleccionadas
